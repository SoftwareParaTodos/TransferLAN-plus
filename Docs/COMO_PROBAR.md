# Cómo probar TransferLAN+ v1.0.1-beta

1. Instalar Go.
2. Ejecutar `run_windows.bat` o `run_linux.sh`.
3. Abrir `http://localhost:5050`.
4. Seleccionar un archivo.
5. Presionar "Subir archivo".
6. Revisar la carpeta `core/server/downloads`.

## Desde el celular

Si la PC y el celular están en la misma Wi-Fi:

1. Ejecutar el servidor en la PC.
2. Mirar la IP local que muestra la consola.
3. En el navegador del celular abrir:

```text
http://IP-DE-LA-PC:5050
```

Ejemplo:

```text
http://192.168.1.45:5050
```

4. Subir un archivo desde el celular.
