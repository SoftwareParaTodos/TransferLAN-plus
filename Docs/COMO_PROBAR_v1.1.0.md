# Cómo probar TransferLAN+ v1.1.0-beta

## Windows
1. Ejecutar TransferLAN+ v1.1.0-beta.
2. Abrir `http://localhost:5050`.
3. Si Android no detecta, ejecutar como administrador:
   `Windows/PERMITIR_FIREWALL_ADMIN.bat`

## Android
1. Generar APK desde GitHub Actions.
2. Instalar APK.
3. Abrir TransferLAN+.
4. Tocar `BUSCAR DISPOSITIVOS`.
5. Seleccionar la PC.
6. Elegir archivo.
7. Enviar.

## Resultado esperado
La PC aparece como tarjeta con nombre real, sistema y versión. La IP queda escondida en configuración manual.
