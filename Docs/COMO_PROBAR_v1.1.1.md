# Cómo probar TransferLAN+ v1.1.1-beta

## Android

1. Generar APK desde GitHub Actions.
2. Instalar APK.
3. Buscar dispositivos.
4. Seleccionar la PC.
5. Elegir un archivo.
6. Enviar.

Durante el envío debe verse:

- porcentaje,
- MB enviados,
- MB totales,
- velocidad aproximada en MB/s.

## Windows

Ejecutar TransferLAN+ normalmente. Si Android no detecta la PC, ejecutar:

`Windows/PERMITIR_FIREWALL_ADMIN.bat`

como administrador.
