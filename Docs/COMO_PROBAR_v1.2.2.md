# Cómo probar TransferLAN+ v1.2.2-beta

## PC

1. Ejecutar TransferLAN+.
2. Abrir:

```text
http://localhost:5050
```

3. Copiar el código/link que aparece en `Emparejar Android`.

## Android

1. Instalar APK nuevo.
2. Abrir TransferLAN+.
3. Tocar `Pegar código de PC`.
4. Pegar el link `transferlan://connect?...`.
5. Guardar.
6. Elegir archivo y enviar.

## Nota

El scanner QR nativo queda desactivado temporalmente porque cerraba la app.
