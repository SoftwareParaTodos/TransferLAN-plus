# Cómo probar TransferLAN+ v1.2.3-beta

## Windows

1. Ejecutar TransferLAN+.
2. Abrir:

```text
http://localhost:5050
```

3. Enviar un archivo desde Android.
4. En la página Windows debe aparecer en `Últimos recibidos`.

## Android

1. Enviar un archivo.
2. Al terminar debe aparecer confirmación clara.
3. Tocar `Ver historial local` para ver envíos recientes.
