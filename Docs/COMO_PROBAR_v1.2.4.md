# Cómo probar TransferLAN+ v1.2.4-beta

## Windows

1. Ejecutar TransferLAN+.
2. Abrir:

```text
http://localhost:5050
```

3. Debe verse un QR real.
4. También debe abrir:

```text
http://localhost:5050/pairing/qr.png
```

## Android

1. Instalar APK nuevo.
2. Tocar `Escanear QR`.
3. Aceptar permiso de cámara.
4. Escanear el QR de Windows.
5. Debe guardar la PC y conectar.

## Fallback

Si el scanner falla:

1. En Windows copiar el código.
2. En Android tocar `Pegar código de PC`.
3. Pegar y guardar.
