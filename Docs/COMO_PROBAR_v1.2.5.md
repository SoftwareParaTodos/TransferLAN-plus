# Cómo probar TransferLAN+ v1.2.5-beta

## Android

Esta versión debe compilar sin ZXing.

## Windows

1. Ejecutar TransferLAN+.
2. Abrir:

```text
http://localhost:5050
```

3. Ver QR real.
4. También probar:

```text
http://localhost:5050/pairing/qr.png
```

## Emparejar

1. Escanear el QR con Cámara/Google Lens.
2. Copiar el link `transferlan://connect?...`.
3. En Android tocar `Pegar código de PC`.
4. Pegar y guardar.
