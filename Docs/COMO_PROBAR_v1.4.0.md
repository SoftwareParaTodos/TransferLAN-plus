# Cómo probar TransferLAN+ v1.4.0-beta

## Android

1. Instalar APK `TransferLANPlus-v1.4.0-beta-debug-apk`.
2. Abrir TransferLAN+.
3. Ver pantalla `Mis dispositivos`.
4. Si hay PC conocida, debe aparecer como tarjeta.
5. Tocar `Enviar archivo`.
6. Elegir archivo.
7. Ver progreso y confirmación.

## Si no hay PC conectada

Usar una opción secundaria:

- Buscar nuevamente.
- Pegar código de PC.
- Agregar por IP.

## Resultado esperado

La IP ya no es protagonista. El usuario ve principalmente la PC/dispositivo.
