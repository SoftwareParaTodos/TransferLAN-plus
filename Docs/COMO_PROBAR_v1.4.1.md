# Cómo probar TransferLAN+ v1.4.1-beta

## Prueba chica

1. Enviar una foto o PDF chico.
2. Debe completar normalmente.

## Prueba grande

1. Enviar archivo de 500 MB.
2. Luego probar archivo cercano a 1 GB.
3. Android no debería cerrarse.
4. La pantalla debe quedar activa.
5. El progreso debe llegar a 99%.
6. Solo debe mostrar 100% cuando Windows confirma.

## Si falla

Debe mostrar error en pantalla, no cerrar la app.
