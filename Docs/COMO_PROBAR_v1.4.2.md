# Cómo probar TransferLAN+ v1.4.2-beta

## Prueba de archivo grande

1. Conectar Android con Windows.
2. Elegir archivo grande.
3. Enviar.
4. Durante el envío debe verse:
   - progreso;
   - destino;
   - archivo;
   - tamaño;
   - estado.

## Si falla

La app no debe cerrarse.

Debe aparecer:
- mensaje de error;
- archivo conservado;
- botón `Reintentar envío`.

Tocar `Reintentar envío` sin volver a elegir archivo.
