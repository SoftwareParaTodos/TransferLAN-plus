# Cómo probar TransferLAN+ v1.5.0-beta

## Prueba de transferencia larga

1. Instalar APK.
2. Aceptar permiso de notificaciones si Android lo pide.
3. Conectar con la PC.
4. Enviar archivo grande.
5. Debe aparecer notificación de TransferLAN+ con progreso.
6. La pantalla debe mantenerse activa.
7. El progreso debe quedar en 99% hasta confirmación.
8. Al terminar, la notificación debe decir transferencia completada.

## Si Android no muestra notificación

En Android 13+ revisar permisos:
Ajustes → Apps → TransferLAN+ → Notificaciones → permitir.
