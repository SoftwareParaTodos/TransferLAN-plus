# Cómo probar TransferLAN+ v1.5.2-beta

1. Instalar APK.
2. Conectar Android con Windows.
3. Elegir archivo.
4. Tocar Enviar.
5. Debe aparecer notificación de TransferLAN+ con progreso.
6. La Activity debe quedar libre.
7. Probar cancelar desde la notificación.

Resultado esperado:
- El archivo llega a Windows.
- La notificación muestra progreso.
- Cancelar corta el envío.
