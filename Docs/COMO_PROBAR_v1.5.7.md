# Cómo probar TransferLAN+ v1.5.7-beta

1. Instalar APK.
2. Conectar con la PC.
3. Tocar `Diagnosticar conexión`.
4. Debe decir conexión correcta si Windows está activo.

## Prueba de error

1. Cerrar TransferLAN+ en Windows.
2. Tocar `Diagnosticar conexión`.
3. Android debe mostrar soluciones claras:
   - abrir TransferLAN+ en Windows;
   - revisar Firewall;
   - revisar Wi‑Fi;
   - verificar IP.
