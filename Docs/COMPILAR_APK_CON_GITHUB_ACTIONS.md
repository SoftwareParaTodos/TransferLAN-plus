# Compilar APK con GitHub Actions

Esta versión incluye un workflow para que GitHub compile el APK automáticamente.

## Pasos

1. Subí esta versión al repositorio.
2. Entrá a GitHub.
3. Abrí la pestaña:

```text
Actions
```

4. Elegí:

```text
Build Android APK
```

5. Tocá:

```text
Run workflow
```

6. Cuando termine, entrá al run y descargá el artefacto:

```text
TransferLANPlus-v1.0.4-beta-debug-apk
```

Adentro vas a tener:

```text
app-debug.apk
```

Ese APK lo podés pasar al celular e instalarlo.

## Importante

Es un APK debug de prueba. Para publicar en Google Play más adelante hay que generar APK/AAB firmado.
