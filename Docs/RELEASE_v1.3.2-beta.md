# Release TransferLAN+ v1.3.2-beta

## Assets sugeridos

- ZIP Windows portable.
- APK generado por GitHub Actions:
  `TransferLANPlus-v1.3.2-beta-debug-apk`.

## Descripción

TransferLAN+ v1.3.2-beta es una versión candidata para pruebas públicas.

Incluye transferencia Android → Windows por red local, servidor Windows portable, QR visible en Windows, emparejamiento por código, agregar por IP, PC conocida, progreso, historial y confirmación final.

## Limitaciones conocidas

- El discovery automático puede fallar en algunos hotspots.
- El scanner QR interno queda pendiente.
- El QR puede usarse con Cámara/Google Lens y luego pegar el código en Android.
