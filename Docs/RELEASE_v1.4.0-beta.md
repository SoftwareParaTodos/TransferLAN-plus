# Release TransferLAN+ v1.4.0-beta

## Resumen

Primera versión con experiencia Android simplificada tipo AirDrop.

## Cambios visibles

- Nueva pantalla `Mis dispositivos`.
- PC conocida como tarjeta.
- Botón principal `Enviar archivo`.
- Opciones técnicas pasan a segundo plano.
- Mantiene transferencia, progreso, historial y fallback por IP/código.

## Limitaciones conocidas

- Discovery UDP puede fallar en algunos hotspots.
- Scanner QR interno sigue pendiente.
