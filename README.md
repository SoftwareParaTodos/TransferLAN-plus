# TransferLAN+ v1.0.7-beta

Sin cuentas. Sin nube. Sin cables.

## Novedades
- Asistente de Firewall para Windows.
- Endpoints `/device/info` y `/network/info`.
- Web local mejorada con IP visible.
- Android con botón "Buscar dispositivos".
- Envío Android -> Windows conservado.

## Prueba rápida
1. Ejecutá TransferLAN+ en Windows.
2. Ejecutá `Windows/PERMITIR_FIREWALL_ADMIN.bat` como administrador si Android no conecta.
3. En Android tocá "Buscar dispositivos".
4. Elegí archivo y enviá.
