TransferLAN+ v1.0.2-beta Portable Windows

COMO USAR:
1) Descomprimir este ZIP.
2) Doble click en INICIAR_TRANSFERLAN.bat o TransferLAN+.exe.
3) Se abre el navegador en http://localhost:5050.
4) Para enviar desde el celular, usar la URL que muestra la pantalla: http://IP-DE-TU-PC:5050

IMPORTANTE:
- PC y celular deben estar en la misma red Wi-Fi/LAN.
- Si Windows Firewall pregunta, permitir acceso en redes privadas.
- Los archivos recibidos quedan en la carpeta Downloads.

ESTADO:
Esta es una beta ejecutable mínima: recibe archivos por navegador local.
No requiere Go, Python ni instalación.
